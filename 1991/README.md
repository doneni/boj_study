# 트리 순회 *Tree Traversal*
![img](./1.png)

트리를 순회하는 방법 3가지
<li> 전위(preorder)
<li> 중위(inorder)
<li> 후위(postorder)

## 전위 순회 *Preorder Traversal*
**root → L → R** 순서로 방문
> A → B → D → C → E → F → G

## 중위 순회 *Inorder Traversal*
**L → root → R** 순서로 방문
> D → B → A → E → C → F → G

## 후위 순회 *Postorder Traversal*
**L → R → root** 순서로 방문
> D → B → E → G → F → C → A
